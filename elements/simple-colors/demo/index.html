simple-colors

Simple-colors is a method for managing colors. It includes a custom element, shared styles, and more.

The styles can be used to set specific colors, or to have the colors dynamically adapt based on the attributes of a custom element. Try adjusting the parent-element below to see this in action.

My background color is always light but the color changes based on my parent's element's accent-color.

My background color is always red, but the shade changes based on whether or not my parent is light or dark.
