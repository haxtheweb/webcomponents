simple-colors Classes

Simple colors CSS classes have the following naming convention: .simple-colors-[theme name]-[base color]-[shade level][-property], as in --simple-colors-default-theme-red-3.

Theme name can be light-theme or dark-theme. Or it can be default-theme, which inherits the custom element's theme.

Base color can be any of the following 18 base colors: grey, red, pink, purple, deep-purple, indigo, blue, light-blue, cyan, teal, green, light-green, lime, yellow, amber, orange, brown, and blue-grey. Or it can be set to accent, which inherits the custom element's accent color.

Shade level can be a number from 1 to 12. Typically, lower shade levels are for backgrounds and higher levels are for foregrounds. to increase color contrast, increase the difference between the foreground and background shade levels.

Classes without a property specified, as in simple-colors-default-theme-red-3, set the background color. Classes with the -text property, as in simple-colors-default-theme-red-3-text, set the text color. Classes with the -border property, as in simple-colors-default-theme-red-3-border, set the border.
