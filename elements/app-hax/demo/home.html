What is HAX?

HAX is a paradigm shift for web publishing. It puts advanced authoring capabilities in the hands of everyone, regardless of technical ability and all items made are sustainability built in highly semantic HTML.

Why HAX?

  • Simplicity - Blocks, Media, Edit system to ensure ease of use
  • Sustainability - Writes HTML + Web components so it will never go out of date!
  • Portability - HAX sites can be downloaded as single ZIP folder and put on any server
  • Community - A diverse network of educators, students and staff driving requirements
  • Accessible - HAX elements and themes are highly accessible and proactively prevent making common A11y mistakes
  • Stylish - HAX themes are decoupled from designed, meaning content isn't tied to presentation.
