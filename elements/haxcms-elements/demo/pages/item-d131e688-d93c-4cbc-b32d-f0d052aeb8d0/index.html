HAX Glossary - Definition List Demo

Below is a definition list showcasing HAX concepts. Try editing this section to test the new keyboard behaviors for DT/DD elements!

HAX
Headless Authoring eXperience - A platform that enables rich web content creation without coding
Web Components
Reusable custom elements that encapsulate functionality and styling
Contenteditable
A web standard that allows users to edit HTML content directly in the browser
Meme Maker
A custom HAX element that creates memes by overlaying text on images
Definition List
An HTML structure using dl, dt, and dd elements to define terms and their meanings

Keyboard Tips: When editing, press Enter on a DT (term) to jump to or create its DD (definition). Press Enter on a DD to create a new DT+DD pair.
