Basic responsive-utility demo

There are two ways to use this utility: on web components or on other elements.

Web Components

To use it web components:

  1. Add it to the component's array of behaviors: behaviors: [ ResponsiveUtilityBehaviors ],
  2. Optional: If you wish to override the default properties:
    this.attribute = "custom-attribute-name-here"; //optional: custom attribute name, default is "responsive-size"
    this.relativeToParent = false; //optional: false for media query behavior; default is true for element query behvavior
    this.sm = 400; //custom sm breakpoint; default is 600
    this.md = 800; //custom md breakpoint; default is 900
    this.lg = 1200; //custom lg breakpoint; default is 1200
    this.xl = 1600; //custom xl breakpoint; default is 1500

Other Elements

To use it on other elements:

  1. Use a script to call the utility:
    <script>
      window.ResponsiveUtility.requestAvailability();
      var el = document.getElementById('element-id-here');
      el.fire('responsive-element',{
        "element": el,
        "attribute": "responsive-size", //optional: custom attribute name, default is "responsive-size"
        "relativeToParent": true, //optional: false for media query behavior; default is true for element query behvavior
        "sm": 400, //optional: custom sm breakpoint; default is 600
        "md": 800, //optional: custom md breakpoint; default is 900
        "lg": 1200, //optional: custom lg breakpoint; default is 1200
        "xl": 1600, //optional: custom xl breakpoint; default is 1500
      });
    </script>
  2. Include styles based on the attribute you set, for example: #element-id-here [responsive-css-attribute-here = "xs"] { display: none; }
