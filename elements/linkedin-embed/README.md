# &lt;linkedin-embed&gt;

Embed
> LinkedIn profile badge embed web component

## Usage
```js
/* In an existing JS module / web component */
import "@haxtheweb/linkedin-embed/linkedin-embed.js";
```

Importing `linkedin-embed.js` registers:
- `linkedin-embed`

## Develop / Demo
Running `npm start` starts a local development server and watches JavaScript and `lib/` files.
```bash
$ npm start
```
